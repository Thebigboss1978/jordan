## ⚡ Quick Start (Local Development)

**Prerequisites:** Node.js

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Configure Environment:**
   Set the required API keys in `.env.local`
3. **Launch the Core:**
   ```bash
   npm run dev
   ```
